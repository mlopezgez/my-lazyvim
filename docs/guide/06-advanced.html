Terminal Integration

Floating Terminal

Run shell commands without leaving Neovim using Snacks terminal.

Keybind Description
Ctrl+/ Toggle floating terminal

Terminal Tips

  • Toggle Behavior: Press Ctrl+/ to open, press again to hide (keeps session alive)
  • Exit to Normal Mode: Press Ctrl+\Ctrl+n to use Vim motions in terminal
  • Persistent: Terminal session persists‚Äîyour shell history is preserved
  • Copy Output: Enter normal mode, yank text with y
‚ö°

Pattern: Quick Shell Commands

  1. While editing code: Ctrl+/
  2. Run command: cargo build, pytest, npm run dev, etc.
  3. See output immediately
  4. If successful: Ctrl+/ to close, continue coding
  5. If errors: copy error text, paste into Claude or fix manually
  6. Re-run: ‚Üë to recall command, Enter
üí° Tip: This is faster than :! for interactive or multi-line output.

Troubleshooting

Rust LSP not working?

  • Ensure rust-analyzer is installed: rustup component add rust-analyzer
  • Check :LspInfo to see if server is running
  • Try restarting LSP: :LspRestart
  • For Cargo projects, ensure Cargo.toml exists in project root

Terminal not responding?

  • Press Ctrl+\Ctrl+n to exit insert mode in terminal
  • Close terminal: :q in terminal window
  • Reopen fresh terminal: Ctrl+/
  • Check for hung processes: ps aux | grep nvim

UI toggles not working?

  • Some toggles are buffer-local, others are global
  • Check :set option? to see current state (e.g., :set spell?)
  • Restart Neovim if toggles are stuck
  • Check :messages for error logs
